How to Embed a YouTube Video

Step-by-step instructions on how to embed any embeddable YouTube video directly into an HTML page. No flash, no download, just copy and paste the embed code. I will be using this royalty-free waterfall nature video as an example below.

Embedding allows you to play YouTube videos directly on your webpage rather than providing a link to YouTube. Here are the five steps I took to embed this video at the bottom of this page. I have included a screenshot of each step.

01

Find and open the video on YouTube

Go to youtube.com and search for the video you want to embed. For this tutorial, I used a short, free-use nature clip called "Free Waterfall 4k Videoclip." Click the video's thumbnail to open it.

youtube.com/results?search_query=free+waterfall+4k+video+clip
02

Click the "Share" button

Below the video player, find the row of icons under the title and click Share (labeled "Paylaş" in this browser's Turkish interface). This opens a menu with several sharing options.

youtube.com/watch?v=w6uX9jamcwQ
03

Select "Embed" from the share menu

A panel of sharing icons appears (Messages, WhatsApp, Facebook, etc.). The first icon — a pair of angle brackets < > — is Embed. Click it.

youtube.com/watch?v=w6uX9jamcwQ
04

Copy the embed code

YouTube generates an <iframe> snippet in a text box, with a Copy button beside it. Click Copy to save that code to your clipboard.

youtube.com/watch?v=w6uX9jamcwQ
05

Paste the code into your HTML

Open your HTML file (locally in your code editor or directly in the GitHub file editor) and paste the copied <iframe> code directly where you would like the video to be shown. Save (or commit) the file and refresh the page in your browser to check that the video displays.

github.com — datacodecrafts-video
embed code 
<iframe width="560" height="315"
  src="https://www.youtube.com/embed/w6uX9jamcwQ"
  title="YouTube video player" frameborder="0"
  allow="accelerometer; autoplay; clipboard-write;
  encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen>
</iframe>
